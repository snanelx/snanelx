# О СЕБЕ

Fullstack-разработчик с опытом почти 6 лет. Делаю веб-сервисы, сайты, Telegram-ботов, мини-приложения для Telegram, backend API, CRM и внутренние инструменты для бизнеса.

Работаю с проектами от идеи и архитектуры до запуска, деплоя и поддержки.

## Чем занимаюсь

- Веб-разработка: сайты, сервисы, кабинеты, CRM, админ-панели (также копирую и дорабатываю чужие проекты)
- Backend-разработка: API, авторизация, базы данных, очереди, интеграции
- Telegram-разработка: боты, mini apps, автоматизация процессов
- Frontend: адаптивные интерфейсы, анимации, интерактивные страницы
- DevOps: Docker, Docker Compose, Nginx, Apache, деплой и настройка окружений

## Стек

**Frontend:** React, TypeScript, Next.js, Vue, Nuxt, Tailwind, SCSS, Sass, Vite, Webpack, Three.js, jQuery

**Backend:** Laravel, NestJS, Fastify, Express, GraphQL, Prisma

**Databases:** PostgreSQL, MySQL, MariaDB, SQLite, MongoDB, Firebase, Redis

**Infrastructure:** Docker, Docker Compose, Nginx, Apache, Kafka, WebSocket

**Telegram:** Telegraf.js, Telegram Bots, Telegram Mini Apps
